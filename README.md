# Centro Studio — Versione Generata con AI
Progetto sperimentale che rappresenta una prima bozza di un centro di controllo dedicato allo studio.
Questa versione è stata interamente generata con strumenti di Intelligenza Artificiale, con l’obiettivo
di analizzare, testare e valutare la qualità del codice prodotto automaticamente.

## Obiettivo
Questa prima stesura serve per:

- osservare come l’AI struttura un layout di dashboard

- valutare componenti, scelte stilistiche e organizzazione del codice

- identificare cosa mantenere, migliorare o riscrivere manualmente

- confrontare questa versione con le altre due realizzate a mano o con supporto AI

## Tecnologie
- HTML5

- CSS3

- Bootstrap 5

- Generazione AI (layout, struttura e contenuti)

## Stato del progetto
Bozza generata automaticamente.
Il codice verrà analizzato e rifinito nelle versioni successive.
